# Bem-vindo de volta ao Raccoon, Bruno!

Este é o seu cérebro estendido - um assistente equilibrador que te dá pistas de memória, sem criar dependência.

## O que já construímos aqui

Desde 15/11/2025, você tem trabalhado para criar um sistema que:

1. **Te equilibra** - estrutura quando você devaneia, flexibilidade quando você trava no detalhe
2. **Respeita sua neurodivergência** - feito para seu jeito de pensar e processar informação
3. **Mantém acessibilidade** - visual contrastado e definido, respeitando seus 15% de visão
4. **Funciona de verdade** - baseado em Notas Mínimas autocontidas, não em redes complexas de links

## Sistema atual

**Fundamentos:**
- Bloco básico: Nota Mínima (autocontida, funcional, não precisa de links excessivos)
- Integração com Claude Code via sistema de memória em `.claude/`
- Repositório Git com regras de commit padronizadas
- Templates e estruturas que facilitam uso sem configuração obsessiva

**Ferramentas ativas:**
- Obsidian para edição e visualização
- Git para versionamento (repositório: github.com/miillerb/racoon-vault)
- Claude Code para assistência técnica com memória persistente

## Valores que guiam este vault

1. **Curiosidade** - explore sem medo de quebrar o sistema
2. **Flexibilidade** - proteção contra foco obsessivo no detalhe errado
3. **Organização** - clareza sobre o que merece atenção
4. **Acessibilidade** - alto contraste e definição visual sempre

## Lembre-se

Quando tiver disperso (devaneando demais):
→ A estrutura te segura - criar/consultar Nota Mínima focada

Quando tiver travado (configurando demais):
→ A flexibilidade te solta - PARAR, usar o que já tem

Quando não souber se algo merece atenção:
→ A organização te guia - verificar se conecta com seus valores
